## 系统理论
```
系统的优劣取决于子系统的优劣以及子系统的相互关系。比如说木桶效应不仅取决于各个板的高低，也取决于
板间的联系，即粘合程度。
```

## 临界点效应
```
99摄氏度的水并不能驱动蒸汽机，而添一把火，100摄氏度的水立刻就能出现大量的水蒸气，从而驱动蒸汽机。

因此做不好小事的人，是很难做成大事的。坚持做好小事到一定程度，"大事"只是水到渠成。
```
